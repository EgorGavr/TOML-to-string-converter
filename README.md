# TOML-to-string-converter
Реализован набор функций для преобразования массива целых чисел в строку формата TOML
и обратно.
На вход функции передаётся массив целых чисел и название массива. Функция
преобразования в строку возвращает строку в формате TOML.
Функция преобразования из строки получает на вход строковое представление
массива и возвращать исходный массив.
Например, массив чисел 1,3,2 при названии массива «array» преобразовывается в
строку:
[array]
1=1
2=3
3=2
Реализованы 3 перегрузки описанных функций:
1. Работает с нуль-терминированной строкой (const char*).
2. Работает с массивом символов и их количеством (const char* + size_t).
3. Работает с экземпляром класса std::string
